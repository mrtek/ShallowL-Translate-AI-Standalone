## 🚀 О проекте
[cite_start]**ShallowL Translate AI Standalone** — это автономное приложение для высококачественного машинного перевода, работающее локально на вашем ПК[cite: 1]. [cite_start]Проект использует мощь нейросети `Dolphin 12B (Mistral Nemo)` и движка `KoboldCPP`, обеспечивая абсолютную конфиденциальность данных и качество перевода, сопоставимое с лучшими онлайн-сервисами[cite: 1].

[cite_start]Ориентирован на пользователей с видеокартами от 8 ГБ VRAM (рекомендуется 12-16 ГБ)[cite: 1].

![Интерфейс программы](gui.png)

## ✨ Основные возможности
* **100% Локально:** Никаких облачных API, подписок или сбора данных. [cite_start]Ваш текст не покидает пределы компьютера[cite: 1].
* **Умные Промпты:** Встроенный редактор стилей. [cite_start]Переводите технические мануалы, художественную литературу, игры (с сохранением тегов) или используйте специализированные стили[cite: 1].
* [cite_start]**Индивидуальные настройки ИИ:** Температура, Top-P и Штраф за повторы привязаны к каждому конкретному промпту для максимальной гибкости[cite: 1].
* **Мультиформатность:** Прямая загрузка `TXT`, `DOCX` и `PDF`. [cite_start]Сохранение готового перевода в `TXT` или `DOCX`[cite: 1].
* [cite_start]**Адаптивный UI:** Поддержка английского и русского языка интерфейса, темная и светлая темы[cite: 1].
* [cite_start]**Умные горячие клавиши:** `Ctrl+C/V/A` работают безупречно на любой раскладке клавиатуры[cite: 1].
* [cite_start]**Гарантированная очистка:** При закрытии программы VRAM видеокарты полностью освобождается от фоновых процессов[cite: 1].

## 🛠️ Установка и Запуск

### Требования
* ОС: Windows 10/11 (64-bit)
* GPU: NVIDIA или AMD (минимум 8 ГБ видеопамяти)
> 💡 **Никакого стороннего софта:** Программа полностью автономна. Установка Python в систему **не требуется**, всё необходимое будет скачано и настроено внутри папки проекта автоматически.

### Шаг 1. Установка
1. Скачайте проект на свой компьютер.
2. Запустите файл `01_install.bat`. Он автоматически:
   * Скачает и настроит портативную версию Python 3.12.
   * Установит все необходимые библиотеки.
   * Скачает ядро `koboldcpp.exe` в папку `bin/`.
   * Скачает модель `Dolphin 12B` в папку `models/`.

### Шаг 2. Запуск
В зависимости от вашей видеокарты, запустите соответствующий файл:
* Для видеокарт NVIDIA: `02_start_NVIDIA.bat`
* Для видеокарт AMD: `02_start_AMD.bat`

[cite_start]Дождитесь появления сообщения **«Нейросеть готова к работе!»** — интерфейс будет разблокирован автоматически[cite: 1].
