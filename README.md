## 🚀 О проекте
**ShallowL Translate AI Standalone** — это автономное приложение для высококачественного машинного перевода, работающее локально на вашем ПК. Проект использует мощь нейросети `Dolphin 12B (Mistral Nemo)` и движка `KoboldCPP`, обеспечивая абсолютную конфиденциальность данных и качество перевода, сопоставимое с лучшими онлайн-сервисами.

Ориентирован на пользователей с видеокартами от 8 ГБ VRAM (рекомендуется 12-16 ГБ).

![Интерфейс программы](gui.png)

## ✨ Основные возможности
* **100% Локально:** Никаких облачных API, подписок или сбора данных. Ваш текст не покидает пределы компьютера.
* **Умные Промпты:** Встроенный редактор стилей. Переводите технические мануалы, художественную литературу, игры (с сохранением тегов) или используйте киберпанк-сленг.
* **Индивидуальные настройки ИИ:** Температура, Top-P и Штраф за повторы привязаны к каждому конкретному промпту для максимальной гибкости.
* **Мультиформатность:** Прямая загрузка `TXT`, `DOCX` и `PDF`. Сохранение готового перевода в `TXT` или `DOCX`.
* **Адаптивный UI:** Поддержка английского и русского языка интерфейса, темная и светлая темы.
* **Умные горячие клавиши:** `Ctrl+C/V/A` работают безупречно на любой раскладке клавиатуры.
* **Гарантированная очистка:** При закрытии программы VRAM видеокарты полностью освобождается от фоновых процессов.

## 🛠️ Установка и Запуск

### Требования
* ОС: Windows 10/11
* Python: 3.12 (или совместимый)
* GPU: NVIDIA или AMD (минимум 8 ГБ видеопамяти)

### Шаг 1. Установка
1. Скачайте проект на свой компьютер.
2. Запустите файл `01_install.bat`. Он автоматически:
   * Создаст изолированное виртуальное окружение.
   * Установит все необходимые библиотеки.
   * Скачает ядро `koboldcpp.exe` в папку `bin/`.
   * Скачает модель `Dolphin 12B` в папку `models/`.

### Шаг 2. Запуск
В зависимости от вашей видеокарты, запустите соответствующий файл:
* Для видеокарт NVIDIA: `02_start_NVIDIA.bat`
* Для видеокарт AMD: `02_start_AMD.bat`

Дождитесь появления сообщения **«Нейросеть готова к работе!»** — интерфейс будет разблокирован автоматически.
